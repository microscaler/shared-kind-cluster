# Auto-research — documentation (observability)

This directory holds the **observability improvement charter** for the shared Kind cluster: control surface, time budget, experiment log, and “do not repeat” list. It is the single place a scheduled agent (or human) records **what was tried** and **what is in scope** for world-class stack observability.

| [`OBSERVABILITY_CONTROL_SURFACE_AND_LOG.md`](./OBSERVABILITY_CONTROL_SURFACE_AND_LOG.md) | Policy, **time budget** for phases A–D, **control surface** (what we may change), **experiment log**, cron checklist |
