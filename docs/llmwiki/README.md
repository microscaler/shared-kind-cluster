# Shared Kind cluster — LLM wiki

Living, LLM-maintained knowledge for the **shared-kind-cluster** repository, per [Karpathy's llm-wiki pattern](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).

**If you are an agent starting a session, read in this order:**

1. [`AGENTS.md`](../../AGENTS.md) — strict rules at the repo root.
2. [`SCHEMA.md`](./SCHEMA.md) — how this wiki is structured and maintained.
3. [`index.md`](./index.md) — catalog of every page.
4. [`log.md`](./log.md) — chronological log; **tail** for recent work by other sessions.
5. [`docs-catalog.md`](./docs-catalog.md) — what formal docs and manifests the wiki reflects.

Then read the **topics** that match the task (cluster layout, observability, etc.).
